TT - Testing Tool Documentation

Creating a new test

  • test_new - create a new test
  • test_new_sta - create a new test within the sta package
  • test_new_sta_3 - create 3 new tests within the sta package
The following information is required for a new test and will be asked for:
  • package (if not provided in original command)
  • test group(s) (existing test groups shown, can be selected by number)
  • ade commands (defaults to: source test.tcl)
  • platform (defaults to: current platform)
  • runtime (no default)
  • ade compiled-in options (defaults to: none)
  • ade startup options (defaults to: none)
Runtime can be omitted just by entering a blank. At minimum, the user must still create/provide two more files:
  • test.tcl - ade commands that produce the test output
  • test.log.GOLD - default "GOLD" file, expected output from running the test
    can be initially created by: make test.log; mv test.log test.log.GOLD

A test requires test.exe (automatically created) which calls (by default) test.tcl. test.tcl should be a series of tcl ade commands which performs the required test. Usually, the success or failure of the test will then be determined by doing a diff of the output vs. test.log.GOLD.
